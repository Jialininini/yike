练习案例:
	今日一刻	
		两天 两遍保留index.html
		项目结构:
		public  ----文件夹 用于存放共用的东西
				 |--js 别人提供的js jquery,angularJS
				 |--css bootstrap
				 |--imgs 如用户注册时默认的头像
				 |--fonts
		uploads	   ---上传文件
		scripts	   ---自己定义的脚本文件
		views 	   ----保存所有的视图
		index.html ----首页

	1. 根据上面的案例结构,调整源文件的路径关系
	2. 将案例文件初始化为本地仓库,在github上新建远程仓库并与本地仓库关联,将本地仓库提交后推送到远程仓库上
	3. 创建angularjs应用
		3.1 引入js
			yike文件夹中右击 git bash ,bower install angular-route 下载并引入
		3.2 定义 app应用名称,创建app模块,并调用run方法
		3.3 找到对应的a标签,添加单击事件,在根作用域上绑定对应的toggle方法,测试toggle可以执行
		3.4 通过观察源文件样式的改变,查找侧边导航栏 导航栏内容,头部,主题发生的改变的原因
		3.5 在根作用域上绑定collapsed属性,默认值为false,点击触发toggle时将该值去翻,实现 隐藏显示的效果
		3.6 观察导航栏内容位移,是通过修改css样式实现的,获取导航栏所有的dd,通过遍历修改dd们的样式
